# Sales Analysis Dashboard using Excel, Power Query & Power Pivot
## Project Objective
The objective of this project is to transform raw sales data into meaningful business insights by applying data cleaning, transformation, modeling, and visualization techniques in Microsoft Excel. The dashboard aims to analyze revenue performance, customer behavior, product trends, city-wise sales, and seasonal patterns to support data-driven decision-making through interactive and dynamic reporting.
